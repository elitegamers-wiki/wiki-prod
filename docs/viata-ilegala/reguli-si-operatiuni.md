## Regulile de aur

### Comportamentul mafiotului

::: warning PRINCIPII FUNDAMENTALE
1. **Respectul este bidirecțional** - oferiți respect pentru a primi respect
2. **Cuvântul dat este lege** - nu promiteți ce nu puteți ține
3. **Familia este sacră** - problemele interne rămân interne
4. **Discreția este putere** - ce știe familia, rămâne în familie
5. **Onoarea peste profit** - nu orice ban merită câștigat
:::

### Limbajul și comunicarea

- Limbajul trebuie să fie **decent** - chiar și criminalii au clasă
- **Toxicitatea** este interzisă și se sancționează sever
- Respectați **toți jucătorii** - adversari, civili, autorități

### Conflicte armate

- În cazul conflictelor armate între mafii și poliție, familia își poate chema ca întăriri doar proprii membri sau membri ai unei familii aliate.

### Loialitatea și trădarea

**Loialitatea:**
- Membrii pun familia înaintea intereselor personale
- Nu se divulgă informații despre operațiuni, membri sau planuri
- Se răspunde la chemările familiei, indiferent de situație

## Operațiuni speciale - Răpirea membrilor de înalt rang

::: warning MISIUNE DE ELITĂ: CAPTURAREA PROTEJATULUI
Această operațiune reprezintă una dintre cele mai riscante și profitabile acțiuni pe care o familie le poate întreprinde. Este o regulă care generează roleplay intens, cu tensiune, strategie și consecințe reale.
:::

### Țintele valide
- **Membri ai Sindicatului**
- **Lideri ai familiilor Neoficiale/Oficiale**

### Regulile operațiunii

::: danger RESTRICȚII DE TIMP ȘI PARTICIPANȚI
- Răpirea se poate efectua **doar între orele 20:00 și 24:00**
- **Minimum 8 membri** din aceeași familie IC
- **Maximum 16 membri** în total implicați în operațiune
:::

**Pregătirea:**
1. Planificați operațiunea cu atenție - fără improvizații
2. Asigurați-vă că aveți un loc sigur pentru reținere
3. Pregătiți echipa conform limitelor de participanți
4. Documentați totul cu GoPro valid

**Execuția:**
1. Capturați ținta folosind tactici de roleplay elaborate
2. Transportați-o la locația sigură
3. Anunțați capturarea prin canalele potrivite
4. Începe cronometrul de **1 oră**

::: warning TRATAMENTUL MEMBRILOR SINDICAT
Membrii Sindicatului **NU** pot fi:
- Jigniți sau insultați
- Răniți sau loviți
- Omorâți sub nicio formă

Încălcarea acestor reguli duce la **sancțiuni severe** pentru întreaga grupare.
:::

**Condiții de succes:**
- Ținta trebuie reținută **minimum 1 oră** fără a fi eliberată
- Dacă cineva reușește să o elibereze în acest interval, operațiunea eșuează
- Nu se acceptă locații inaccesibile sau exploit-uri

### Negocierile și răscumpărarea

**Dacă ținta este Membru Sindicat:**

::: info PROTOCOLUL RĂSCUMPĂRĂRII
- Răpitorii negociază direct cu Sindicatul
- Suma se stabilește în funcție de rangul țintei
- Plata este suportată de:
  - Cei care **NU** au reușit să recupereze membrul
  - Cei care **NU** au răspuns la Codul Roșu
- Refuzul de plată = consecințe diplomatice severe
:::

**Dacă ținta este Lider de Familie:**
- Negocierile se poartă cu familia țintei
- Suma poate fi mai mare sau mai mică, în funcție de relațiile existente
- Se pot negocia și alte avantaje pe lângă bani

### Codul Roșu - Protocolul de urgență

Când un membru de rang înalt este răpit:

1. **Alertă imediată** - se anunță toate forțele disponibile
2. **Mobilizare totală** - toți membrii disponibili trebuie să răspundă
3. **Coordonare rapidă** - se stabilește planul de salvare
4. **Execuție precisă** - timpul este esențial

**Consecințele nerespectării Codului Roșu:**
- Cei care nu răspund vor suporta partea lor din răscumpărare
- Absențele repetate pot duce la sancțiuni interne
- Lipsa de loialitate nu este tolerată

## Relația cu Sindicatul

### Ce este Sindicatul

Sindicatul reprezintă **autoritatea supremă** în lumea criminală. Este format din cele mai influente și experimentate figuri ale orașului și are rolul de:
- A menține ordinea între organizații
- A media conflictele
- A stabili regulile jocului
- A aproba sau respinge noi organizații
- A sancționa abaterile
