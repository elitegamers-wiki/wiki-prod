<h3>Ce este sistemul de drone?</h3>

Drona este un dispozitiv de zbor. Cat timp piloteaza drona, personajul tau ramane pe loc cu telecomanda in mana, iar tu vezi totul prin camera dronei. Este folosita pentru recunoastere, filmat, supraveghere sau pentru a observa zone greu accesibile.

<h3>De ce am nevoie ca sa pilotez?</h3>

<ul>
  <li><b>Licenta de zbor</b> — fara o licenta de zbor valida nu poti porni nicio drona. O cumperi din magazinul EG (Store). Licenta este legata de personajul tau si are o data de expirare.</li>
  <li><b>O drona in inventar</b> — un item de tip drona pe care il folosesti din inventar pentru a o lansa.</li>
</ul>

<h3>Tipuri de drone</h3>

  <thead>
    <tr>

<h3>Cum o folosesc?</h3>

<ul>
  <li>Deschide inventarul si <b>foloseste itemul de drona</b>.</li>
  <li>Personajul scoate telecomanda, iar drona este lansata in fata ta.</li>
  <li>Camera trece automat pe drona si apare interfata de zbor (HUD).</li>
</ul>

<b>Atentie:</b> nu poti lansa drona daca esti intr-un vehicul sau intr-un interior. Iesi afara, intr-un spatiu deschis.

<h3>Comenzi de zbor</h3>

<ul>
  <li><span class="eg-drone-key">W</span> Inainte &nbsp; <span class="eg-drone-key">S</span> Inapoi</li>
  <li><span class="eg-drone-key">A</span> Stanga &nbsp; <span class="eg-drone-key">D</span> Dreapta</li>
  <li><span class="eg-drone-key">Q</span> Roteste stanga &nbsp; <span class="eg-drone-key">E</span> Roteste dreapta</li>
  <li><span class="eg-drone-key">Shift</span> Urca &nbsp; <span class="eg-drone-key">Ctrl</span> Coboara</li>
  <li><span class="eg-drone-key">Space</span> Zoom camera</li>
  <li><span class="eg-drone-key">↑</span> Inclina camera in sus &nbsp; <span class="eg-drone-key">↓</span> Inclina camera in jos</li>
  <li><span class="eg-drone-key">H</span> Ascunde sau arata HUD-ul</li>
</ul>

Lista completa o gasesti oricand in joc apasand butonul <b>Instructiuni</b> din interfata dronei.

<h3>Interfata de zbor (HUD)</h3>

<ul>
  <li><b>Date de zbor</b> — viteza, altitudine si distanta fata de tine.</li>
  <li><b>Busola</b> — directia in care este orientata drona.</li>
  <li><b>Status</b> — nivelul bateriei, semnalul si coordonatele.</li>
  <li><b>Screenshot</b> — fa o poza prin camera dronei; aceasta se salveaza automat in galeria telefonului.</li>
  <li><b>Night Vision</b> — disponibil doar la drona cu vedere pe timp de noapte.</li>
</ul>

<h3>Baterie</h3>

<ul>
  <li>Fiecare drona are o baterie care se consuma cat timp zbori.</li>
  <li>Nivelul ramas este <b>salvat pe item</b> — daca recuperezi drona, data viitoare porneste de unde ai ramas.</li>
  <li>Cand bateria este aproape goala primesti un avertisment — intoarce-te repede ca sa o recuperezi.</li>
  <li>Daca bateria se descarca complet in zbor, <b>drona se prabuseste</b> si o pierzi.</li>
</ul>

<h3>Raza de actiune si avarii</h3>

<ul>
  <li>Drona poate zbura doar pana la o anumita distanta de tine. Daca te apropii de limita, primesti un avertisment; daca o depasesti, <b>pierzi controlul</b>.</li>
  <li>Drona are puncte de viata. La impacturi puternice se avariaza, iar dupa prea multe lovituri <b>se distruge</b>.</li>
  <li>Drona FPV este cea mai fragila — piloteaza-o cu grija.</li>
</ul>

<h3>Cum recuperez drona?</h3>

<ul>
  <li>Pentru a opri drona in siguranta, <b>adu-o aproape de tine</b> si apoi dezactiveaz-o. Itemul iti revine in inventar cu bateria ramasa.</li>
  <li>Daca dezactivezi drona <b>departe de tine</b>, ea se distruge si pierzi itemul.</li>
  <li>Daca drona se prabuseste (baterie goala, avarii sau iesire din raza), itemul este pierdut.</li>
</ul>

<h3>De unde cumpar o drona dupa ce am achizitionat licenta de zbor?</h3>

<ul>
  <li>Aceasta poate fi cumparata de la un magazin de tip DigitalDen cu bani in joc.</li>
</ul>
